### Recursive/Droste Effect

**Concept:** Infinite loop with subject holding device showing same scene

**Template:**
```
Recursive image of [SUBJECT: orange cat, person, object] [POSITION: sitting in office chair, standing, posed] holding up [DEVICE: iPad, picture frame, mirror].

On the [DEVICE] is the same [SUBJECT] in the same scene holding up the same [DEVICE].

Rules:
- Each iteration should be clearly visible for at least 3-4 levels deep
- Maintain perfect alignment and perspective matching
- Each nested image should be proportionally smaller
- Same lighting and color grade throughout
- Sharp focus on outermost iteration, progressive softness going inward

Technical:
- High resolution to show detail in nested images
- Proper perspective matching at each level
- Consistent lighting across all iterations
- Clean edges on device screens
- Professional composition

Quality: 4K, sharp detail, perfect recursive alignment
```

## Split View 3D Renders

### Half-Realistic Half-Wireframe

**Concept:** Product render with realistic left half, wireframe right half

**Template:**
```
Create a high-quality realistic 3D render of exactly one instance of: [PRODUCT: Orange iPhone 17 Pro, wireless headphones, smartwatch]

Position: Float freely in mid-air, gently tilted and rotated in 3D (not front-facing)
Background: Soft minimalist dark background, 1080×1080 composition

Left Half — Full Realism:
- Appears exactly as in real life
- Accurate materials, colors, textures, reflections, proportions
- Completely opaque, no transparency, no wireframe overlay
- No soft transition, no fading, no blending

Right Half — Hard Cut Wireframe Interior:
- Clean switch to wireframe interior diagram
- Boundary: Perfectly vertical, perfectly sharp, crisp cut line (top to bottom)
- No diagonal edges, no curved slicing, no gradient

Wireframe Specifications:
- Use only two line colors:
  * Primary: white (≈80% of lines)
  * Secondary: color sampled from dominant product color (<20% of lines)
- Lines: thin, precise, aligned, engineering-style
- Every component must perfectly match geometry
- Show internal components accurately

Strict Rules:
- Render ONLY ONE object in entire frame
- No second object from any angle
- No reflections/shadows/silhouettes showing second object
- No duplicate objects or mirrored pairings

Pose & Lighting:
- Natural subtle tilt + rotation
- Soft neutral global illumination
- No shadows under object
- No extra props or text

Quality: Clean, professional product visualization
```
